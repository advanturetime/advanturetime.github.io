  • 01

    Card Game

    "Vanilla JS"로 제작된 카드 뒤집기 게임입니다. JSON을 사용하여 코드의 가독성을 높였으며 타이머의 기능 구현과 게임 진행에 필요한 예외 처리 (카드는 2개 이상으로 뒤집어 지지 않는다, 일정한 시간 차를 두어 카드를 뒤집게 한다 등) 에 중점을 두어 프로젝트를 진행하였습니다.

    CLICK ME
  • 02

    Music Player

    "Vanilla JS" 로 제작된 뮤직 플레이어입니다. 프로젝트를 진행하며 audio 객체의 다양한 메서드를 다룰 수 있게 되었습니다 (pause, duration 등). input type = "range"를 사용하여 범위 내에서 드래그가 가능한 플레이어를 만들었으며 네이티브 기능만을 사용한 커스텀 디자인을 통해 짧고 간결한 코드의 즐거움을 알게 되었습니다.

  • 03

    Movie Reservation

    "JavaScript" 를 사용해 영화 예매 사이트를 재현해보았습니다. 이미 예약된 좌석은 빨간색으로, 예약 가능한 좌석은 회색으로 표현했으며, 선택한 좌석은 눈에 가장 잘 들어오는 파란색 계열로 정했습니다. 다양한 쿼리 선택자를 사용함으로써 요소로의 접근을 용이하게 하는것에 초점을 두었습니다.

  • 04

    Slider

    "Vanilla JS" 로 무한루프 슬라이드를 제작하였습니다.
    box-reflect과 transition 효과를 이용해 자연스러운 넘김 효과를 주었습니다. 이미지의 사이즈는 Sass의 변수값을 사용하여 유지보수성, 재사용성을 높였습니다. 조건문은 삼항 연산자를 사용하여 효율적이고 간결하게 작성해 보았습니다.

    CLICK ME
  • 05

    Vue

    "Vue" 로 제작된 투두리스트입니다. 리스트는 v-for 반복문을 사용하여 간결하고 깔끔하게 DOM을 생성하였습니다. App.vue에 각 컴포넌트를 import 시켰으며 methods와 이벤트를 사용하여 리스트의 상태 관리를 구현해보았습니다.

  • 06

    React

    "React"를 사용해 Create(생성), Update(갱신), Delete(삭제) 기능이 구현된 리스트를 만들었습니다. JSX를 사용해 친숙하고 표현력이 뛰어난 코드를 작성하였으며, 적절한 콜백함수를 통해 컴포넌트 사이의 부드러운 데이터 연결을 구현하였습니다.

2020.01.28 ~ 2021.08.17 KBANK PROJECT

1년 8개월 간의 Kbank 고도화 및 유지·보수 프로젝트를 통해 디자이너, 백엔드 개발자와 존중하며 원활한 협업을 끌어내는 방법을 익히게 되었습니다. Sass, Zeplin의 사용법을 배웠고 유지·보수를 고려한 코드 작성 방식을 알게 되었습니다. 기술은 사람을 위해 존재하고, 결국은 누군가를 더욱 행복하게 해주는 것이 개발의 최종 목표라고 생각합니다. 목표 실현을 위해 먼저 좋은 사람이 되어, 올바른 가치관을 가지고 사람들과 소통하며, 사용자에게 최상의 서비스를 제공하기 위해 최선을 다하겠습니다.

※ 금융권 보안상의 이유로 코드를 첨부할 수 없어 캡처로 대신합니다 ※
